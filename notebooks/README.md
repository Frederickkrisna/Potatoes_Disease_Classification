# Jupyter Notebooks Experiments

Folder ini berisi catatan eksperimen, eksplorasi data, dan prototyping sebelum kode dipindahkan ke modul produksi (`src/`).

## Daftar Notebook

1.  **`exploration.ipynb`**
    * Exploratory Data Analysis (EDA).
    * Visualisasi distribusi kelas.
    * Cek sampel gambar dan statistik dataset.

2.  **`experiment.ipynb`**
    * Tempat uji coba training model awal.
    * Tuning hyperparameter manual.
    * Visualisasi Loss dan Accuracy curve.

3.  **`evaluate_checkpoint.ipynb`**
    * Analisis mendalam hasil model terbaik.
    * Membuat Confusion Matrix dan Classification Report.
